User Interface Design

What is UI?

User interfaces are access points from where users can interact with designs in order to achieve specific goals. click here for more.

Purpose of User Interface Design

To focus the design on building highly usable and efficient interfaces, enabling the user to detatch from the machine and achieve goals succesfully, directly, and effortlessly

What makes a UI great?

The combination of aesthetic components such as symmetry, proximity, colour theory, proportional distribution, amongst others

Differences between UI and UX

While user interaction focuses on the use, accesibility and functionality of the design, a well implemented user experience can be achieved by the use of feedback, resulting in a fully immersive process for the user.click here for more
